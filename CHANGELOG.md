# Changelog

## v2.1.0

**New Features:**

### Generative Mode
Automatic evolving harmonies that mutate while you hold a chord.

**How to access:**
- Press **Button 11** to open Special Modes menu
- Rotate encoder to select "Generative"
- Press encoder to enter settings

**Settings:**
- **Speed** (0-100%): How fast chords evolve. 0=slow mutations, 100=rapid changes
- **Type**:
  - *Chords*: Hops between pads using musical intervals (thirds, fourths, fifths)
  - *Scales*: Stays on same pad, morphs between related scale types

**Controls in settings:**
- Rotate encoder = change value
- Click encoder = next setting (Speed → Type → exit)
- Button 11 = exit menu

**Using it:**
1. Enable Generative mode and exit menu
2. Hold any chord pad
3. Watch/hear the harmony evolve automatically
4. LED flashes cyan on each mutation

### Screensaver
Starfield animation after 30 seconds idle. Any input exits.

### Play Chords (Arp Settings)
- Access: **Shift + Arp+** to open Arp Settings, go to page 8
- When ON: Plays full chord alongside arpeggiated notes

### Button Changes
- **Button 11**: Now opens Special Modes (was Arp Settings)
- **Shift + Arp+**: Now opens Arp Settings

**Improvements:**
- Expanded to 32 scale types (added Ukrainian Dorian, Lydian Augmented, Bebop scales, Neapolitan, Enigmatic, Hirajoshi)
- 16 style preset banks (added Neo-Soul, Rock, Blues, Latin, Cinema, Trap)
- Default max notes increased to 8
- Improved encoder button responsiveness
